# Kickoff Timing

## Intent
Kickoff questions are currently asked before starting work on project assets, but at that point there may be no Plan yet. Asking review cadence and version bump before tasks exist puts the cart before the horse. Kickoff questions should be asked when the user approves the Plan, not when work begins.
