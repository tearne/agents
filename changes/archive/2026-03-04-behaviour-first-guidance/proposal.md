# Proposal: Behaviour-First Guidance
**Status: Draft**

## Intent
Notes and drafts may naturally be expressed in technical or implementation terms — that is often how intent first forms. But before a proposal is finalised, guide the user towards expressing required behaviours instead. Reserve technical specification for genuine constraints or non-functional requirements.

## Specification Deltas

### ADDED
