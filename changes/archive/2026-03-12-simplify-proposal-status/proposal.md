# Proposal: Simplify Proposal Status
**Status: Approved**

## Intent
The `Ready for Review` status adds a step without adding value — in practice, the user approves a proposal directly from `Draft`. Removing it simplifies the status lifecycle.

## Specification Deltas

### MODIFIED
- Proposal status progression simplified from `Note → Draft → Ready for Review → Approved` to `Note → Draft → Approved`
