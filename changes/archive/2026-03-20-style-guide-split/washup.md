# Washup: Style Guide Split

## Intent
Validate whether a craft guide can be usefully split into a terse agent-optimised rules file and a companion elaboration file, and whether maintaining both is worth the overhead.

## Outcome
The split is successful and worth adopting. The two-file pattern is clean, the elaboration file complements rather than duplicates the rules file, and the naming convention (`*-elaboration.md`) is self-describing.

## Spec Deltas
None applied directly. The findings and conventions established here feed into the structural proposal under development (`changes/open/structure-exploration/`), which will apply this pattern across all craft guides as part of a broader reorganisation.
