# Additional Guides
This directory lists guides that are not always loaded. Name one in a spec or proposal to opt in; the agent will then apply it.

| Guide | Description | Loading |
|-------|-------------|---------|
| `VERSIONING.md` | Semantic versioning conventions | Prompted — agent raises when setting up a project, changes are accumulating, or a breaking change is being made |
| `POS.md` | Python Orchestrated Script style | Passive — name it when writing a POS-style script |

Companion files named `*-elaboration.md` contain examples and references for their guide. The agent need not load them unless additional context is needed. `STYLE-elaboration.md` is the companion to `AGENT/STYLE.md`.
